# Parameters

The minting module contains the following parameters:

| Key           | Type            | Example                |
|---------------|-----------------|------------------------|
| MintDenom     | string          | "utbd"                 |
| InflationRate | string (dec)    | "0.130000000000000000" |
| BlocksPerYear | string (uint64) | "6311520"              |
